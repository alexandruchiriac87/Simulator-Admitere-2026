# 🎓 Simulator Admitere Liceu

Un simulator interactiv pentru admiterea la liceu în București și Brașov.

Aplicația permite elevilor și părinților să estimeze șansele de admitere folosind media de la Evaluarea Națională și ultimele medii oficiale de admitere.

## Funcționalități

- 🔍 căutare rapidă după liceu
- 📊 filtrare după profil și specializare
- 📈 comparație cu ultimele medii de admitere
- 📋 construire fișă de opțiuni
- 🖨 export pentru tipărire
- 📱 interfață responsive

## Date utilizate

- ultimele medii oficiale de admitere
- rezultatele Evaluării Naționale 2026
- statistici pentru București și Ilfov

## Tehnologii

- HTML5
- CSS3
- Vanilla JavaScript


---

Acest proiect a fost realizat ca proiect personal pentru a ajuta elevii și părinții în procesul de admitere la liceu.
